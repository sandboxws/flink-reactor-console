# Next steps

You're set up. From here:

- **Synthesize** SQL + CRDs from the terminal:
  ```bash
  flink-reactor synth
  ```
- **Author faster** with the starter snippets — type `frpipeline`, `frsource`,
  `frsink`, `frtransform`, or `frschema` in a `.tsx` pipeline.
- **Tune** the language server under **Settings → Extensions → FlinkReactor**
  (`flinkReactor.server.debounce`, `flinkReactor.flinkVersion`, …).

Learn more at the [FlinkReactor documentation](https://flink-reactor.dev).
